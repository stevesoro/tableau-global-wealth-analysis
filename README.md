# Global Wealth Analysis Dashboard (Tableau)

## Overview
This project explores wealth distribution across countries in 2019.  
The objective is to highlight global disparities and provide a clear, interactive view of how wealth is concentrated geographically.

🔗 **Live Dashboard (Tableau Public):**  
https://public.tableau.com/app/profile/yedjoyety.soro/viz/Assignment_WealthbyCountry2019/WealthbyCountry2019Map?publish=yes

---

## Business Question
- How is global wealth distributed across countries?
- Which countries dominate global wealth?
- What geographic patterns emerge from wealth concentration?

---

## Key Insights
- Wealth is highly concentrated in a small number of countries.
- Significant disparities exist between regions.
- Some countries with large populations do not correspond to high wealth levels.
